# ABOUT

Little tool to visualize the GCODES commands from your 3D printer, CNC, laser, etc.
Drop the .gcode from your slicer (like UltiMaker Cura, Prusa Slicer, etc) and let the application to parse the gcode to produce analytical information like extruded volume, max speed, etc.
If you have a custom built 3D printer, this can help to detect issues or maximize printing speed.
Project scafoolded with Vite.
Created: Mar 2025

# HOW TO RUN

npm install
npm run dev
